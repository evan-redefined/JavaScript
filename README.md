Core Functions of kalkulator.js

    Mathematical Expression Evaluation: Capable of processing text-based input for standard arithmetic operations, including addition (+), subtraction (-), multiplication (*), and division (/).

    Support for Grouping Operations: Supports the use of parentheses () to define the order of precedence in complex calculations.

    Interactive Interface: Utilizes the readline module to create a continuous prompt experience, allowing users to perform multiple calculations without having to restart the script.

    Error Handling: Features a try...catch mechanism to intercept invalid inputs and prevent the program from crashing unexpectedly due to formatting errors.
